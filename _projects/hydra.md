Hydra was first presented in {% cite Hughes22rss-hydra %}, a spatial perception system that is capable of building 3D scene graphs from sensor data in real-time.
A 3D scene graph is a powerful high-level representation of 3D environments that describes the environment as a layered graph where nodes represent spatial concepts at multiple levels of abstraction and edges represent relations between concepts. 

Please check out our [repo](https://github.com/MIT-SPARK/Hydra).

Please refer to {% cite Hughes23arxiv-hydraFoundations %} for the detailed description of the system.
